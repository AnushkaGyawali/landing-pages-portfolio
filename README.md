## How to Run Locally

```bash
# Clone the repository
git clone https://github.com/anushkagyawali/landing-pages-portfolio.git

# Navigate into the folder
cd landing-pages-portfolio

# Open the portfolio showcase
open index.html
```

No build tools, no npm, no dependencies. Every file is plain HTML and opens directly in any browser.

---

## How to Deploy on GitHub Pages

```
1. Go to your repository on GitHub
2. Click Settings
3. Click Pages in the left sidebar
4. Under Source select: Deploy from a branch
5. Select branch: main
6. Select folder: / (root)
7. Click Save
8. Wait 2 to 3 minutes
9. Your site is live
```
